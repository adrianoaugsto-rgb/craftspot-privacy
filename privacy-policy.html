Política de Privacidade

Última atualização: maio de 2026  ·  Versão 1.0

1. Introdução

O CraftSpot é um aplicativo desenvolvido para ajudar artesãos a registrar, organizar e gerenciar seus trabalhos com fotos e localização. Esta política explica como tratamos os dados dos nossos usuários.

2. Dados Coletados

O CraftSpot pode coletar ou acessar os seguintes dados:

  • Fotos e imagens dos trabalhos cadastrados pelo usuário
  • Localização geográfica (quando autorizado pelo usuário)
  • Informações de conta criadas pelo usuário (nome, e-mail via Firebase)
  • Dados de uso do aplicativo para fins de melhoria

3. Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Exibir e organizar os trabalhos cadastrados pelo próprio usuário
  • Mostrar localização dos trabalhos no mapa
  • Sincronizar dados entre dispositivos via Firebase
  • Melhorar a experiência de uso do aplicativo

Não vendemos, compartilhamos ou cedemos seus dados a terceiros para fins comerciais.

4. Armazenamento e Segurança

Os dados são armazenados de forma segura utilizando os serviços do Firebase (Google). Aplicamos medidas técnicas razoáveis para proteger suas informações contra acesso não autorizado.

5. Permissões do Dispositivo

O aplicativo pode solicitar as seguintes permissões:

  • Câmera e Fotos: para adicionar imagens aos trabalhos
  • Localização: para registrar e exibir a localização dos trabalhos no mapa
  • Contatos: não acessamos seus contatos

Você pode revogar qualquer permissão a qualquer momento nas configurações do seu dispositivo.

6. Seus Direitos

Você tem o direito de:

  • Acessar os dados que armazenamos sobre você
  • Solicitar a exclusão da sua conta e dados
  • Exportar seus dados

7. Crianças

O CraftSpot não é direcionado a menores de 13 anos e não coleta intencionalmente dados de crianças.
